| Perintah | fungsi |
|---       |---     |
| su       | beralih ke root|

| pwd | melihat lokasi(directory) di mana kita berada saat ini |
| mkdir (nama directory) | membuat directory baru|
| rmdir (nama directory) | menghapus directory |
