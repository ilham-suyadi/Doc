# Tag

## Create Tag

```sh
git tag <tag name>
```

## SHow Tag

```sh
git tag
```

## Push Tag ke Github

```sh
git push origin <tag name>
```

## Pull Tag di Github ke local

```sh
git pull
```

## Pindah ke Tag Lain

```sh
git checkout <tag name tujuan>
```

## Pindah kembali ke semula

```sh
git checkout main
```
