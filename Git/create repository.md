# Git Repository

## Create New Repository

untuk membuat sebuah repository mneggunakan git, bisa menggunakan perintah :

```sh
git init <name repository>
```

## Clone Repository

perintah mengeclone sebuah repository menggunakan git adalah :

```sh
git clone <url>
```
